struct CondChecker<'a> {
    parser: &'a Parser<'a>,
    forbid_let_reason: Option<ForbiddenLetReason>,
    missing_let: Option<MaybeMissingLet>,
    comparison: Option<MaybeComparison>,
}
Expand description

Visitor to check for invalid/unstable use of ExprKind::Let that can’t easily be caught in parsing. For example:

ⓘ
// Only know that the let isn't allowed once the `||` token is reached
if let Some(x) = y || true {}
// Only know that the let isn't allowed once the second `=` token is reached.
if let Some(x) = y && z = 1 {}
