Expand description

A gen/kill dataflow problem.

Each method in this trait has a corresponding one in Analysis. However, the first two methods here only allow modification of the dataflow state via “gen” and “kill” operations. By defining transfer functions for each statement in this way, the transfer function for an entire basic block can be computed efficiently. The remaining methods match up with Analysis exactly.

Analysis is automatically implemented for all implementers of GenKillAnalysis via a blanket impl below.
