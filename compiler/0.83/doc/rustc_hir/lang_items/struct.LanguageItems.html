rustc_hir::lang_items

Struct LanguageItems

source 
pub struct LanguageItems {
    items: [Option<DefId>; 192],
    reverse_items: FxIndexMap<DefId, LangItem>,
    pub missing: Vec<LangItem>,
}
Expand description

All of the lang items, defined or not. Defined lang items can come from the current crate or its dependencies.

Fields§

§items: [Option<DefId>; 192]

Mappings from lang items to their possibly found DefIds. The index corresponds to the order in LangItem.

§reverse_items: FxIndexMap<DefId, LangItem>§missing: Vec<LangItem>

Lang items that were not found during collection.
