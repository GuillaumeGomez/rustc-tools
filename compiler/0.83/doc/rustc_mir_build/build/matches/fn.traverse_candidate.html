Expand description

A depth-first traversal of the Candidate and all of its recursive subcandidates.

This signature is very generic, to support traversing candidate trees by reference or by value, and to allow a mutable “context” to be shared by the traversal callbacks. Most traversals can use the simpler Candidate::visit_leaves wrapper instead.
