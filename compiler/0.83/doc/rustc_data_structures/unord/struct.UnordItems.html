pub struct UnordItems<T, I: Iterator<Item = T>>(I);
Expand description

UnordItems is the order-less version of Iterator. It only contains methods that donโt (easily) expose an ordering of the underlying items.

Most methods take an Fn where the Iterator-version takes an FnMut. This is to reduce the risk of accidentally leaking the internal order via the closure environment. Otherwise one could easily do something like

โ
let mut ordered = vec![];
unordered_items.all(|x| ordered.push(x));

Itโs still possible to do the same thing with an Fn by using interior mutability, but the chance of doing it accidentally is reduced.
